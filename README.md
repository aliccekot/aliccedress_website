# aliccedress - Интернет-магазин одежды

Современный интернет-магазин одежды с адаптивным дизайном и удобным интерфейсом для покупок.

## Описание проекта

Проект представляет собой полнофункциональный интернет-магазин одежды, построенный на современном стеке технологий. Проект разделен на две основные части:

- **ui-library/** - библиотека переиспользуемых React компонентов
- **frontend/** - основное приложение интернет-магазина

### Основные возможности
- Каталог товаров с фильтрацией
- Детальные карточки товаров

### Компоненты библиотеки

#### 1. ProductCard
Умная карточка товара с полной информацией:
- Изображение товара с обработкой ошибок загрузки
- Название, описание и цена
- Кнопка "Подробнее" 
- Кнопка добавления в корзину
- Гибкие настройки отображения текста
- Поддержка различных позиций контента

#### 2. Catalog
Гибкий каталог товаров:
- Адаптивная сетка товаров
- Настройка количества колонок
- Обработка кликов по товарам
- Интеграция с корзиной
- Поддержка различных макетов

## Технологии

- **Frontend**: React 19 + TypeScript
- **Сборка**: Vite
- **Тестирование**: Jest + React Testing Library
- **Стилизация**: CSS Modules
- **Линтинг**: ESLint с строгими правилами кодстайла
- **Разработка**: Hot Reload, TypeScript проверка


### Предварительные требования
- Node.js 18 или выше
- npm 

### Установка и запуск

1. **Клонирование репозитория**

git clone <repository-url>
cd aliccedress_website

2. **Установка зависимостей UI Library**
cd ui-library
npm install

3. **Сборка компонентов**
npm run build

4. **Запуск тестов**
npm run test

5. **Запуск демо-приложения**
cd ../frontend
npm install
npm run dev

6. **Открытие в браузере**
Приложение будет доступно по адресу: http://localhost:5173

## Скрипты проекта
### UI Library (ui-library/)
- npm run build - сборка библиотеки
- npm run dev - разработка с вотчером
- npm run test - запуск тестов
- npm run lint - проверка кодстайла
- npm run test:coverage - запуск тестов с генерацией отчета о покрытии кода

### Frontend (frontend/)
- npm run dev - запуск dev-сервера
- npm run build - сборка для production
- npm run test - запуск тестов
- npm run lint - проверка кодстайла
